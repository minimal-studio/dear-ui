# Helper

内置一些工具函数

## 获取 Icons 的 mapper

```js
import { getIcon, getIconMapper } from '@deer-ui/core/config'
```

## 动态加载脚本 LoadScript

```js
import { LoadStuff, LoadScript, LoadLink } from '@deer-ui/core/utils'

// 通过动态标签加载资源
LoadStuff({ src, onload, type = script || css })

// 通过动态标签加载 script
LoadScript({ src, onload })

// 通过动态标签加载 css
LoadLink({ src, onload })
```

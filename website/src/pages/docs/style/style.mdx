# Style

> deer-ui 使用 scss 构建 style，原子分布，逐层构建组件、模版、页面、系统。

## Usage

在项目入口 `style.scss` 文件中 `@import '@deer-ui/core/default.scss'`

```scss
// 默认的颜色变量
@import '@deer-ui/style/color/default.scss';
```

## Custom theme

可以通过覆盖原本 scss 变量定制主题，[详细定义参考](https://github.com/minimal-studio/deer-ui/blob/master/packages/core/style/base/variable.scss)

```scss
// 依赖颜色变量的主样式入口
@import '@deer-ui/style/var.scss';

// 通过后置覆盖原有的变量
@import './custom/var.scss';

// 引用 index.scss 之前需要先引入 var.scss 设置对应的 scss 变量
@import '@deer-ui/style/index.scss';
```

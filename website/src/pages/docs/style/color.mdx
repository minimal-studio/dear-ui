## Colors

`deer-ui` 提供了 `8` 个颜色，每种颜色再按照饱和度细分 `6` 级，对应的规则是 `color-level`, 例如

```scss
$blue;
$blue-1;
$blue-2;
$blue-3;
$blue-4;
$blue-5;
$blue-6;
```

## Usage

### Via className

```jsx live noInline
import { Label, Grid } from '@deer-ui/core';
import { 
  ExampleColors, NatureColors,
  GrayscaleColors, StatusColors,
  ForEachColor
} from '@deer-ui/core/utils';

function Example() {
  const colorLevel = [1,2,3,4,5,6];
  return (
    <Grid container space={0}>
      {
        ForEachColor(NatureColors)((color, idx) => {
          return (
            <Grid key={color} item lg={6} md={6}>
              <Grid container space={5}>
                <Grid item lg={12} md={12}>
                  <div className={`flex bg_${color}`}
                    style={{
                      height: 100
                    }}>
                  </div>
                </Grid>
                {
                  colorLevel.map((_, i) => {
                    const currColor = `bg_${color}-${_}`;
                    return (
                      <Grid key={currColor} item lg={2} md={2}>
                        <div
                          style={{
                            display: 'block'
                          }}
                          className={currColor + ' flex'}
                          style={{
                            height: 100,
                          }}>
                        </div>
                      </Grid>
                    )
                  })
                }
                <div style={{
                  fontSize: 18,
                  marginBottom: 20,
                }}>{color}</div>
              </Grid>
            </Grid>
          )
        })
      }
    </Grid>
  )
}
render(<Example />);
```

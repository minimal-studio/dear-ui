# Icons

建议使用 `react-icons`

## Intro

> 默认使用 fontawesome 的 icons，使用 Icon 组件时默认添加了 fa fa- 前缀，只需要填写对应的图标名即可

- 如果需要切换图表库，可以通过 setUIConfig 设置 icon 的前缀
- 同时支持 icon 的名字映射，通过 setUIConfig 设置 iconMapper

## Setup CDN Recourse

```html
<html>
<head></head>
<body>
  ...
  <link rel="stylesheet" href="https://use.fontawesome.com/releases/v5.11.1/css/all.css">
</body>
</html>
```

## 内置组件的 IconMapper

```js static
import { setUIConfig } from '@deer-ui';

// 使用的默认 icon
setUIConfig({
  iconMapper: {
    'arrow-left': 'angle-left',
    'arrow-right': 'angle-right',
    date: 'calendar-alt',
    more: 'equals',
    close: 'times',
    noData: 'thermometer-empty',
    success: 'check',
    question: 'question-circle',
    error: 'exclamation-circle',
    'circle-up': 'arrow-circle-up',
    'circle-down': 'arrow-circle-down',
    'arrow': 'arrow',
  },
  // icon 的前缀, 关于 iconStyle 详情查看 https://fontawesome.com 描述
  iconPrefix: (iconStyle) => `fa${iconStyle} fa-`
});
```

## Component Usage

```js
<Icon n="balance-scale" />
```

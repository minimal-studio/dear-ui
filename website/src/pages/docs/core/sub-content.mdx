# SubContent

隐藏/显示

## Basic Usage

```jsx live noInline
import { SubContent } from '@deer-ui/core'

function Example() {
  return (
    <SubContent displayElem={'鼠标移至此处出现隐藏的内容'}>
      显示的内容
    </SubContent>
  )
}
render(<Example />);
```

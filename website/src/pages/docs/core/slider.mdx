# Slider

> 滑动选择器

## Basic Usage

```jsx live noInline
import { Slider } from '@deer-ui/core'

function Example() {
  return (
    <>
      <h4>范围 0 ~ 100</h4>
      <Slider
        range={[0, 100]}
        withInput={false}
        onChange={val => {
        }
      } />
    </>
  )
}
render(<Example />);
```

# InputNumber

> 带输入验证的 InputNumber 控件

## 数字范围限制

```jsx live noInline
import { InputNumber } from '@deer-ui/core';

function Example() {
  return (
    <>
      <h4>输入范围 1 ~ 100</h4>
      <InputNumber
        numRange={[1, 100]}
        style={{width: 200}}
        selectable={false}
        onChange={val => {
          console.log(val)
        }} />
    </>
  )
}
render(<Example />);
```

# Container

> 用于外层的容器

## Basic Usage

```jsx live noInline
import { Container } from '@deer-ui/core'

function Example() {
  return (
    <Container
      className="bg_default p20"
    >
      <div>
        Container
      </div>
    </Container>
  )
}
render(<Example />);
```

## Size


## Fluid Container

100% width

```jsx live noInline
import { Container } from '@deer-ui/core'

function Example() {
  return (
    <Container
      fluid
      className="bg_default p20"
    >
      <div>
        Container
      </div>
    </Container>
  )
}
render(<Example />);
```

## 设置 container 的默认 width

通过设置 scss 变量更改 container 的宽度

```scss
$container-sm-width: 576px;
$container-md-width: 768px;
$container-lg-width: 992px;
$container-xl-width: 1200px;
```

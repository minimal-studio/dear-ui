# Input

> 文本输入增强，提供输入验证支持

## Basic Usage

```jsx live noInline
import { Input } from '@deer-ui/core'

function Example() {
  return (
    <Input
      placeholder="请输入"
      onChange={e => console.log(e)}/>
  )
}
render(<Example />);
```

## Inline title

> 为 Input 添加 inline title，与 placeholder 的区别是 title 并不会在输入内容后消失

```jsx live noInline
import { Input } from '@deer-ui/core'

function Example() {
  return (
    <Input 
      title="固定 Title" 
      placeholder="placeholder"
      onChange={e => console.log(e)}/>
  )
}
render(<Example />);
```

## Output number

```jsx live noInline
import { Input } from '@deer-ui/core'

function Example() {
  return (
    <Input 
      title="输出 number 类型" 
      placeholder="输出数字" 
      outputType="number" 
      propsForInput={{
        type: 'number'
      }}
      onChange={e => console.log(e)}/>
  )
}
render(<Example />);
```

## Filter

```jsx live noInline
import { Input } from '@deer-ui/core'

function Example() {
  return (
    <Input 
      title="只能输入数字" 
      filter={val => (val + '').replace(/[^0-9]/g, '')} />
  )
}
render(<Example />);
```

## With Button

```jsx live noInline
import { Input } from '@deer-ui/core'

function Example() {
  return (
    <>
      <Input
        title="带按钮的输入框"
        outputType="number"
        inputBtnConfig={{
          action: (e) => console.log(e),
          text: '按钮'
        }}/>
      <hr />
      <Input
        title="带按钮的输入框"
        outputType="number"
        inputBtnConfig={{
          action: (e) => console.log(e),
          icon: 'search',
          text: ''
        }}/>
    </>
  )
}
render(<Example />);
```

## Rise up title when focus

```jsx live noInline
import { Input } from '@deer-ui/core'

function Example() {
  return (
    <Input n="phone" title="电话号码" placeholder="请输入" riseTitle />
  )
}
render(<Example />);
```

## Block

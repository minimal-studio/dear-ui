# DatetimePicker

> 基于 Flatpickr 的封装，提供时间选择

## Setup

### 样式引入

```js
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/flatpickr/dist/themes/material_blue.css" />
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/flatpickr/dist/themes/material_red.css" />
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/flatpickr/dist/themes/material_orange.css" />
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/flatpickr/dist/themes/airbnb.css" />
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/flatpickr/dist/themes/dark.css" />
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/flatpickr/dist/themes/confetti.css" />
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/flatpickr/dist/themes/flatpickr.css" />
```

> 参考 https://flatpickr.js.org/themes/

## 单个时间选择

### 默认 UTC 格式的时间对象

```jsx live noInline
import { useState } from 'react';
import { DatetimePicker } from '@deer-ui/core';

function Example() {
  const [date, setDate] = useState('');
  return (
    <div>
      <DatetimePicker
        onChange={val => {
          setDate(val)
        }} />
      <hr />
      Output: {date.toString()}
    </div>
  )
}
render(<Example />);
```

### 带时分秒的字符串


## 时间范围选择

### Addon

可以将 DateShortcut 作为 addon 嵌入

```jsx live noInline
import { useState } from 'react';
import { DatetimePicker, DateShortcut, Icon } from '@deer-ui/core';

const renderShortcutAddon = ({ setDate }) => (
  <DateShortcut
    position="right"
    onChange={(val) => {
      setDate(val);
    }}
  >
    <Icon n="dateShortcut" />
  </DateShortcut>
);

function Example() {
  const [date, setDate] = useState('');
  return (
    <div>
      <DatetimePicker
        needTime
        outputAsString={true}
        mode="range"
        addons={renderShortcutAddon}
        onChange={val => {
          setDate(val)
        }} />
      <hr />
      Output: {date.toString()}
    </div>
  )
}
render(<Example />);
```

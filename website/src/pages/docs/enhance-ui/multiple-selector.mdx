# Multiple

> 输入提示控件

## Basic Usage

```jsx live noInline
import { Multiple } from '@deer-ui/enhance-ui';

function Example() {
  return (
    <div style={{width: 160, position: 'relative'}}>
      <Multiple
        onChange={val => {
          console.log(val)
        }}
        min={0}
        max={10000}
        range={[1, 10, 20, 30]} />
    </div>
  )
}
render(<Example />);
```
